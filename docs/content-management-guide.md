## 🖼️ 이미지 콘텐츠 가이드

### **이미지 요구사항**

#### **제품 이미지**
- **크기**: 1200x800px (16:9 비율)
- **형식**: JPG, PNG, WebP
- **파일 크기**: 최대 500KB
- **품질**: 고해상도, 선명한 이미지

#### **회사 소개 이미지**
- **크기**: 1920x1080px (16:9 비율)
- **형식**: JPG, PNG
- **파일 크기**: 최대 1MB
- **품질**: 브랜드 일관성 유지

#### **로고 이미지**
- **크기**: 300x150px (2:1 비율)
- **형식**: PNG (투명 배경)
- **파일 크기**: 최대 100KB
- **품질**: 벡터 기반 권장

### **이미지 파일명 규칙**
```
[카테고리]_[제품명]_[버전]_[크기].[확장자]

예시:
- product_clarus_tech_v1_1200x800.jpg
- company_illutech_intro_v1_1920x1080.png
- logo_texcom_main_v1_300x150.png
```

## 📝 텍스트 콘텐츠 가이드

### **텍스트 작성 규칙**

#### **제목 (Title)**
- **길이**: 50자 이내
- **스타일**: 명확하고 간결
- **키워드**: SEO 최적화

#### **설명 (Description)**
- **길이**: 150-200자
- **스타일**: 읽기 쉬운 문장
- **구조**: 핵심 정보 우선

#### **상세 내용 (Content)**
- **길이**: 500-1000자
- **스타일**: 전문적이면서 이해하기 쉬운
- **구조**: 단락별 구분

### **SEO 최적화 가이드**

#### **메타 태그**
```html
<title>정호그룹 - [제품명] | [핵심 키워드]</title>
<meta name="description" content="[제품 설명 150자 이내]">
<meta name="keywords" content="[키워드1, 키워드2, 키워드3]">
```

#### **키워드 우선순위**
1. **주요 키워드**: 제품명, 브랜드명
2. **보조 키워드**: 기술명, 서비스명
3. **롱테일 키워드**: 구체적인 용도, 특징
